### ¿Qué es HTML?

**HTML** (HyperText Markup Language) es el lenguaje que define la **estructura** del contenido en una página web. No se encarga del diseño (eso es trabajo de CSS), sino de organizar la información: encabezados, párrafos, listas, enlaces, imágenes, etc.

---

### Inicio del documento

Todo documento HTML debe comenzar con la siguiente línea, que indica al navegador que se trata de una página en HTML5:

```html
<!DOCTYPE html>
```

---

### Estructura básica de un documento HTML

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Mi Página Web</title>
  </head>
  <body>
    <!-- Aquí va el contenido visible de la página -->
  </body>
</html>
```

- `<head>`: contiene metadatos, el título y enlaces a estilos, scripts, etc.

- `<body>`: contiene todo lo que se verá en la página (texto, imágenes, botones, etc.)


---

### Etiquetas comunes en el `<body>`

- `<h1>` a `<h6>` → Encabezados, de mayor a menor importancia.

- `<p>` → Párrafos de texto. No importa el formato en el código; el navegador ajusta automáticamente.

- `<pre>` → Texto preformateado que respeta saltos de línea y espacios.

- `<br>` → Salto de línea.

- `<hr>` → Línea horizontal.

- `<!-- comentario -->` → Comentarios en el código (no se muestran en la página).


#### Ejemplo:

```html
<body>
  <h1>Bienvenido a mi web</h1>
  <h3>Introducción</h3>

  <hr>

  <p>
    Esto es un párrafo de ejemplo. El navegador lo mostrará como bloque de texto.
  </p>

  <br>

  <pre>
Este texto
  conserva el
    formato tal cual
  lo escribimos.
  </pre>

  <!-- Esto es un comentario en HTML -->
</body>
```

---

### Estructura semántica: `header`, `main`, `footer`

HTML5 introdujo etiquetas semánticas que mejoran la accesibilidad, el SEO y la organización del código. En lugar de usar solo `<div>`, es recomendable usar:

- **`<header>`**  
    Va al inicio del documento o de una sección. Suele contener el logo, menú, buscador, etc.

- **`<main>`**  
    Solo uno por página. Contiene el contenido principal que justifica la existencia del sitio.

- **`<footer>`**  
    Va al final de la página o sección. Incluye derechos de autor, enlaces legales, contacto, etc.


#### Ejemplo:

```html
<body>
  <header>
    <h1>Mi sitio web</h1>
    <nav>
      <ul>
        <li><a href="#">Inicio</a></li>
        <li><a href="#">Blog</a></li>
        <li><a href="#">Contacto</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <article>
      <h2>Última entrada</h2>
      <p>Este es el contenido principal del sitio.</p>
    </article>
  </main>

  <footer>
    © 2025 • Todos los derechos reservados • <a href="#">Aviso legal</a>
  </footer>
</body>
```
